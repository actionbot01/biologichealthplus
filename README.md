# BiologicHealthPlus — Specialty Pharmacy Website

NABP-accredited specialty pharmacy providing FDA-approved biologic medications for psoriasis, rheumatoid arthritis, Crohn's disease, and ulcerative colitis.

## Tech Stack
Pure HTML · CSS · Vanilla JavaScript — no build tools, no frameworks.

## Files
| File | Purpose |
|------|---------|
| `index.html` | Homepage — all sections, payment modals, navigation |
| `style.css` | All styles — design tokens, mobile-first responsive grids |
| `main.js` | All interactivity — mega nav, mobile drawer, payment modals, animations |
| `netlify.toml` | Netlify deployment config |
| `images/` | Medication product photos |
| `medications/` | Individual medication pages |
| `conditions/` | Condition-specific pages |

## Deploy to Netlify via GitHub
1. Push this folder to a GitHub repository
2. Go to [app.netlify.com](https://app.netlify.com) → **Add new site → Import from Git**
3. Connect your GitHub account and select this repo
4. Netlify auto-detects `netlify.toml` — build settings are already configured
5. Click **Deploy site**
